About Me

I'm James, a second year BSc Computer Science student at the University of Dundee. My main computing interests currently involve low level programming.

My main skills at present are knowledge of C++, and low level knowledge of the x86 instruction set and its binary encoding.

Like so many other kids, I played games on computers from a young age. This eventually led me to discover Scratch, which really kick started my interest in programming. I started off with toy projects to gain an understanding of programming fundementals before learning Python as part of Computer Science in high school.

Learning Python made me want more performance and a more rigid structure in a program, so quite quickly, I decided to learn C++ as well. This led to many projects, the most useful to date being an assembler for x86/x64.
